# Desafio 1 - Biblioteca

## Descrição
Programa simples para registrar informações de leitores em uma biblioteca.

## Funcionalidades
- Coleta nome e sobrenome do usuário
- Coleta idade do usuário
- Exibe as informações em uma única frase formatada

## Requisitos
- Ruby 2.7+

## Como usar
```bash
ruby desafio.rb
```

Digite seu nome, sobrenome e idade quando solicitado. O programa exibirá suas informações no formato:
```
[Nome] [Sobrenome] possui [Idade] anos
```
